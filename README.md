# Career Compass AI

Career Compass AI is a web application designed to help students discover suitable career paths based on their interests, aptitude, and learning behavior.

The platform collects simple inputs from students such as study habits, subject preferences, and aptitude test responses to provide guidance on possible career directions.

## Idea Behind the Project

Many students face difficulty deciding what career path to follow after school.
This project aims to provide a simple digital platform where students can understand their strengths and explore career options that match their abilities.

## Main Features

* Student profile creation
* Daily study input tracking
* Career aptitude test
* AI-based career suggestions
* Simple progress dashboard

## Technologies Used

Frontend:

* React
* TypeScript
* Tailwind CSS

Backend:

* Node.js
* Express.js

Database:

* MongoDB

Visualization:

* Recharts

## Goal of the Project

The goal of this project is to build a practical tool that can help students make better career decisions using data-driven insights and basic AI logic.
